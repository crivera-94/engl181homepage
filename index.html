RF Tracker

A Modular Low-Cost Location Tracking Solution

View project on GitHub

What does it do?

The RF location tracking system provides a modular solution to any short to medium range detection need. With compact modules, this sytem can be included in just about any multi-entity groupings. Using Bluetooth Smart, communication is not only internet independent, but can easily be used to send data to other Bluetooth Smart enabled devices.

RF vs GPS

The goal of this solution is not to replace GPS, but rather work in tandem with it to increase location tracking resolution. By using low cost antenna modules, tracking any number of close proximity entities is easy, precise, and self-reliant.

Drone Use

Drone technology has progressed quite alot. Using RF technology, we aim to take drones one step further and improve multi-drone cluster coordination for both consumer and commercial use. The modular system provides a location tracking enhancement and a medium to facilitate location communication.

Communication

Messaging your friends is really simple when near a wifi network or in an area with data coverage. When these technologies are not accessible, the communication protocol option used in our system is the optimal solution. Bluetooth Smart, communication between nearby peers is easy and doesn't require the use of large-scale external components.
